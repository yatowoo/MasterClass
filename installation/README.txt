ALICE Master Class Installation

**** linux/MacOsx
 1. download the installation script  startVM.sh from githuh: https://github.com/yschutz/MasterClass/releases/tag/0.1  
 2. make it executable: chmod +x startVM.sh
 3. run it: ./startVM.sh
 This will open an instance of a VirtualBox machine. When started, open a terminal in the VM and follow the instructions.

 **** Windows
 1. download the installation script startVM.bat from github: https://github.com/yschutz/MasterClass/releases/tag/0.1  
 2. run it: startVM.bat
 This will open an instance of a VirtualBox machine. When started, open a terminal in the VM and follow the instructions.
