ALICE Master Class Installation

**** linux/MacOsx
 1. download the installation script from github: https://raw.githubusercontent.com/yschutz/MasterClass/master/installation/startVM.sh  
 2. make it executable: chmod +x startVM.sh
 3. run it: ./startVM.sh
 This will open an instance of a VirtualBox machine. When started, open a terminal in the VM and follow the instructions.

 **** Windows
 1. download the installation script from github: https://raw.githubusercontent.com/yschutz/MasterClass/master/installation/startVM.bat  
 2. run it: startVM.bat
 This will open an instance of a VirtualBox machine. When started, open a terminal in the VM and follow the instructions.
